# CriCodecs
CriWare codecs for Node.js. Uses Rust WASM modules for fast audio decoding.

## Supports
- **Node:** v18.0.0+
- **Extracting:**
    - ACB
    - AWB/AFS2
- **Decoding:**
    - HCA
- **Building/Encoding:** Not supported yet.
